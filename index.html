建议避免
本周主题：清淡护肝 + 控糖 + 护肾
建议先用这一周做基础调整：
• 每餐有粗粮（燕麦/糙米/小米其中一种）
• 每天至少一餐有鱼或豆腐
• 每天 1500–1800ml 温水
❌ 完全不能吃（为肝、肾、血糖好）
奶茶 / 珍珠 汽水 / 果汁饮料 蛋糕 / 饼干 / 糖果 油条 / 炸鸡 / 薯条 香肠 / 腊肉 / 咸鱼 猪肝 / 鸡肝 / 内脏 辣火锅 / 烧烤 啤酒 / 白酒 / 红酒 冰淇淋 榴莲（糖+热量高）
⚠️ 需要控制分量
🍚 白米饭：每餐最多半碗，必须混糙米一起煮
🎃 南瓜：一次最多 100g，要配蛋白质（鱼/蛋/豆腐）一起吃
🌽 甜玉米：一次最多半根
🥕 胡萝卜：熟炒量不要太多
🫘 花生：一次最多 10 粒（汤里喝前撇油）
☕ 茶/咖啡：不要配餐喝，饭后 1 小时再喝
🍇 荔枝、葡萄、芒果：少吃，当成偶尔尝一口
🍚 白粥：GI 很高，不建议常喝，改成杂粮粥
每天小目标：
• 喝够温水 1500–1800ml
• 蔬菜量 ≥ 主食量
• 每天不吃任何甜食 / 油炸 / 酒精
